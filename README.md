哈理工autoAC：
使用教程：
在cmd中执行 
	
	java -jar autoAC.jar [start] [end]
start 和 end 两个参数分别表示 题号的起始
如：
	
	java -jar autoAC.jar 1000 10003

务必autoAC.jar  和  autoAC.properties放在同一个文件夹下
务必cmd打开后切换到 以上两个文件所在的目录


杭电AC教程：
执行 

	java -jar autoAC.jar [杭电用户名] [杭电密码] [开始题号] [结束题号]
